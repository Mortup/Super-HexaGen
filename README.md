# Tarea2-CC3501
## Super Hexa Gon

Para ejecutar el juego se necesita la libreria pygame  
Usando el comando python Main.py el juego comienza  

### Controles:
#### En el juego
Flecha Izquierda / Flecha Derecha: Mover  
R: Reiniciar nivel  
ESC: Cerrar juego  
  
#### En el menu:
Flecha Izuiqerda/Derecha/Arriba/Abajo: Mover puntero  
ENTER: Seleccionar  
BORRAR: Volver atras
